分支

在版本控制中，同时推进多个任务，为每个任务，我们就可以创建单独的每个任务的单独分支。

命令名称 作用
git branch 分支名 创建分支
git branch -v 查看分支
git checkout 分支名 切换分支
git merge 分支名 把指定分支合并到当前分支上
