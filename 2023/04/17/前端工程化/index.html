Ajax

Asynchronous JavaScript And XML, 异步的JavaScript和XML

作用：

  • 数据交互：通过Ajax可以给服务器发送请求，并获取服务器响应的数据
  • 异步交互：可以在不重新加载整个页面的情况下，与服务器交换数据并更新部分网页的技术，

Axios

Axios对原生Ajax进行封装，简化书写，快速开发

  • 发送get请求

YApi

  • YApi是一款高效易用功能强大的api管理平台，旨在为开发、产品、测试人员提供接口管理服务

前端工程化

准备

vue-cli

  • Vue-cli是Vue官方提供的一个脚手架，用于快速生成一个Vue的项目模板

  • Vue-cli提供了以下功能

    • 统一的目录结构
    • 本地调试
    • 热部署
    • 单元测试
    • 集成打包上线

  • 依赖环境：nodejs

Vue项目创建

  • 命令行：vue create vue-project01
  • 图形化界面：vue ui

更改端口号

可以在vue.config.js中设置devServer:{ port:7000 }来更改端口号

Vue的组件文件以.vue结尾，每个组件由三个部分组成：<template>、<script>、<style>

Vue组件库Element

element是饿了么团队开发的一套为开发者设计师和产品经理准备的基于Vue2.0的桌面端组件

组件：组成网页的部件，例如超链接、按钮、图片、表格、表单、分页条等等

导入&&组件

新建vue，撰写完成后在App.vue通过<element-view>标签引入

  • pagination 分页：当数据量过多，使用分页分解数据
事件名称 说明 回调参数
size-change pageSize 改变时会触发 每页条数
current-change currentPage 改变时会触发 当前页

  • dialog对话框：在保留当前页面状态的情况下，告知用户并承载相关操作

  • Form表单：由输入框、选择器、单选框、多选框等控件组成，用以收集、校验和提交数据

  • 通过JSOM.stringify(this.form)可以将js的对象转换为JSON格式的字符串

  • 在Vue项目中使用Axios需要在项目目录下安装axios：npm install axios，需要使用axios时，导入axios：import axios from 'axios'

Vue路由

  • Vue Router是Vue的官方路由

  • 组成

    • VueRouter：路由器类，根据路由请求在路由视图中动态渲染选中的组件
    • <router-link>：请求连接组件，浏览器会解析成<a>
    • <router-view>：动态视图组件，用来渲染展示与路由路径对应的组件

  • 可以通过redirect属性来重定向路径防止访问到空页面

打包部署

部署

  • Nginx是一款轻量级的Web服务器/反向代理服务器以及电子邮件（IMAP/POP3）代理服务器，其特点是占有内存少，并发能力强，在各大互联网公司都有广泛的应用
